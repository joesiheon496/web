`<tr>`행 (table row)
`<td>`열 (table datacell)
`<thead>`,`<th>` 테이블 헤더
`<tbody>`
```html
<table>
  <thead>
    <th></th>
    <td></td>
    <td></td>
  </thead>
  <tbody>
    <th></th>
    <td></td>
    <td></td>
  </tbody> 
  <tfoot>
    <th></th>
    <td></td>
    <td></td>
  </tfoot>
</table>
```
테이블 스타일링
```css
table {
  border : 1px solid red;
  border-collapse:collapse; <!--간격 없앰-->
  border-spacing:15px <!-- 간격 넓힘 -->
}

th, td {
  border:1px solid blue;
}

thead th, thead td {
  border 1px solid green;
}
```
